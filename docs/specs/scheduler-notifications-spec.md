# Scheduler and Notification Templates

## Jobs
- Daily due soon summary: decisions with due dates <= 7 days.
- Weekly roadmap nudge: scheduled items nearing target date without recent check-in.
- Daily period boundary check: create next period and apply rollover when needed.

## Notification Templates
- Due soon:
  - Subject: "Upcoming family decisions this week"
  - Body: list title, owner, due date, current status.
- Roadmap nudge:
  - Subject: "Roadmap status update needed"
  - Body: ask for progress and blockers.
- Rollover notice:
  - Subject: "New period started"
  - Body: include new allocations and rollover amounts.
